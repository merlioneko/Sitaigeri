# 機能
主要となる機能について

## 小説作成
- 以下の様な流れで何度も繰り返して小説を作成する
    - ユーザからアイデアを受け取る
    - *ユーザのアイデアを膨らませる*
    - *アイデアから設定を考える*
    - *シーン構造を考える*
    - *各シーンごとに生成*
- 細かく修正を行える様にする

### アイデア精査
- 入力されたアイデアプロンプトの精査
- 他に面白そうなアイデアの提示

### アイデア構造化
- 入力されたアイデアプロンプトを、各設定やシーンに分ける
- 分けられたシーンは更に分けて構造化

### シーン詳細化
- 構造化されたシーンを詳細に書く
- 前のシーンの状態を受ける

### シーン結合
- 各シーンを検証し、結合する

## 小説評価
作成された小説を評価する

### 評価モード
各評価モードはパーソナリティを持つ
- 筆者：この作品の魅力などを作者の目線で説明
- 編集者：誤字脱字・構成・表現等について向上を意図して提案
- 批評家：作品のテーマ、美学、その他について網羅的に考察・評価
- スレッド/SNS：匿名多数による意見の形式

## 小説データ管理
- 小説本文
- 小説の設定資料
- 単語、用語の記録

# UI
機能に対してUIの作りかた

## メインプラットフォーム
- 新規作成ボタン -> 小説編集画面（新規）
- 既存作品から新規作成ボタン -> 小説編集画面（流用新規）
- 既存作品リスト
    - 既存作品選択ボタン -> 作品プラットフォーム（選択された作品）
- 共用用語辞書ボタン -> 辞書プラットフォーム（共用）

## 小説編集画面
### メイングループ
- 題名テキストボックス：ユーザ・LLM可変
- ユーザー案テキストボックス：ユーザ可変、必須項目
- アイデア創出ボタン -> {LLMにユーザー案を渡し、アイデアリストに追加}
- アイデア精査ボタン -> {LLMにアイデアリストを渡し、不要なものを捨てる}
- アイデア結合ボタン -> {アイデアを結合させ草案テキストボックスに渡す}
- アイデアリスト：各アイテムはユーザ・LLM可変
    - アイデアテキストボックス：ユーザ可変
    - アイデア削除ボタン -> {このアイテムを削除}
- 草案テキストボックス：LLM不変

### リストグループ
- キャラクター追加ボタン -> キャラクター編集画面（新規作成）
- キャラクターリスト
    - キャラクター名テキストボックス
    - キャラクター選択ボタン -> キャラクター編集画面(選択されたキャラクター)
    - キャラクター削除ボタン -> {このキャラクターを削除し、関連のシーンを検索}
- シーン追加ボタン -> シーン編集画面（新規作成）
- シーンリスト
    - シーン名テキストボックス：ユーザ・LLM可変
    - シーン選択ボタン -> シーン編集画面（選択されたシーン）
    - シーン削除ボタン -> {このアイテムを削除}
    - 並び替えボタン: ドラッグして位置を入れ替え
- 用語辞書ボタン -> 辞書プラットフォーム（個別/共用）

## キャラクター編集画面
- キャラクター名テキストボックス：ユーザ・LLM可変
- 役割テキストボックス：ユーザ・LLM可変
- 詳細テキストボックス：ユーザ・LLM可変

## シーン編集画面
- シーン名テキストボックス：可変
